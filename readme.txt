Dependencies
========================================
Make sure you have this libraries installed before compiling.

SDL image (lSDL2_image) 
SDL mixer (lSDL2_mixer)
SDL ttf (lSDL2_ttf)

========================================


Installation instructions
========================================
	- Navigate to source code.
	- Command: 'make'
	
========================================

Run
========================================
	- Command: './main'

========================================

Controllers
========================================
	Move Player
		Up: w/↑
		Down: s/↓
		Left: a/←
		Right: d/→
		Shoot: mouse (m1)
	
	General
		Mute/unmute music: m

========================================
